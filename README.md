## Features

- [ ] EPub
- [ ] Utf-8
- [ ] Gb2312
- [ ] Latin-1
- [ ] Font size, line height
- [ ] Themes
- [ ] Day mode, night mode
- [ ] Book resources
- [ ] Local search
- [ ] Online search
- [ ] Import local books
- [ ] Import books by wifi
- [ ] Advertisement
- [ ] Friends chat
- [ ] Share books
- [ ] Text to speech
- [ ] User auth
- [ ] Cloud

## Usage
```
git clone https://github.com/creatint/light
flutter packages get
flutter run
flutter build apk --release
```
